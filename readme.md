## **How to install and run locally ?**
</div>

- npm i
- setup .env file
- write as follow:-\
  port 5000\
  JWT_SECRET=Your JWT Secret\
  MONGO_URI=mongodb+srv://username:password@cluster0.mongodb.net/databaseName
- to use nodemon (npm run dev) otherwise (npm start)
